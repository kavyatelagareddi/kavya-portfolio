Stock Price Prediction Web Application

Project Overview

A machine learning–based web application that predicts next-day stock prices using historical market data. The system compares LSTM and GRU models and provides visual insights through an interactive Flask interface.

Key Features

  • Supports multiple Indian stocks (NSE)
  • Time-series prediction using LSTM and GRU models
  • Historical price visualization using Matplotlib
  • Web interface built with Flask

Machine Learning Approach

  • Sliding window time-series dataset (60-day lookback)
  • MinMax scaling for data normalization
  • 80–20 train-test split
  • Evaluation using RMSE and MAPE

Tech Stack

  • Python, TensorFlow (Keras)
  • Flask, yfinance
  • Pandas, NumPy, scikit-learn
  • Matplotlib
